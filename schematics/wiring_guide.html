⚠️ CRITICAL — 3.3V WARNING The ESP32 runs on 3.3V logic. NEVER connect the 5V pin to the ESP32. Always use the Pro Micro's 3.3V output pin. Using 5V will permanently damage the ESP32.
⚠️ TX/RX MUST BE CROSSED TX on Pro Micro connects to RX on ESP32. RX on Pro Micro connects to TX on ESP32. If you connect TX→TX and RX→RX it will not work.

05 — Assembly Steps
1

Flash ESP32 First

Upload the ESP32 WiFi server code via Arduino IDE before wiring anything. Board: ESP32 Dev Module. Connect ESP32 directly to PC via its USB port.

2

Flash Pro Micro Second

Upload the Pro Micro keylogger code via Arduino IDE. Board: SparkFun Pro Micro 5V 16MHz. Connect via its Micro-USB port.

3

Wire USB-A Female → USB Host Shield

Solder 4 short wires (~3cm) from the breakout board pads to the USB Host Shield pads: VBUS, D−, D+, GND. This is the only soldering needed.

4

Connect USB Host Shield → Pro Micro

Use female-female Dupont wires. Connect all 6 SPI pins as per Connection 1 table above. Double-check VCC=5V, GND, SS=Pin10, MOSI=Pin16, MISO=Pin14, SCK=Pin15.

5

Connect ESP32 → Pro Micro

Use female-female Dupont wires. Connect as per Connection 2 table. Remember: 3.3V only, TX crossed to RX, RX crossed to TX.

6

Plug Keyboard into Female USB Port

The USB keyboard cable plugs into the USB-A female breakout board. This is the input side.

7

Plug Pro Micro into PC

The Pro Micro's built-in Micro-USB port connects to the target PC. The PC will detect it as a keyboard automatically.

8

View Logs Over WiFi

On your phone or laptop connect to WiFi: KeyloggerAP password password123. Open browser and go to http://192.168.4.1 to see all keystrokes.
